# Olympus Design System

Welcome to the Olympus design system, inspired by 100% Hex aesthetic.

## Philosophy

Our design system follows Hex's core design principles:

- **Data-first design** - Results and visualizations are primary, UI chrome is secondary
- **Professional tool aesthetic** - Serious, business-oriented tone for enterprise users
- **Conversational AI integration** - Natural language as first-class input method
- **Source transparency** - Clear visual indicators for SQL vs document results
- **Mobile-first responsive** - Adaptive layouts with touch-friendly targets

## Color Palette

Our color system is built on Hex's visual language:

- **Primary Blue** (`#4B7FFF`) - Main brand color for CTAs, links, and active states
- **Accent Purple** (`#8B5CF6`) - AI features, highlights, and special functionality
- **Neutral Grays** - Complete scale from `#F9FAFB` (backgrounds) to `#111827` (darkest text)
- **Semantic Colors** - Success (`#10B981`), Error (`#EF4444`), Warning (`#F97316`)

### Source Badge Gradients

Visual indicators for hybrid query results:

- **SQL Results** - Blue gradient: `linear-gradient(to right, #4B7FFF, #3366FF)`
- **Document Citations** - Green/teal gradient: `linear-gradient(to right, #10B981, #0D9488)`
- **Computations** - Purple gradient: `linear-gradient(to right, #8B5CF6, #7C3AED)`

## Typography

- **Interface Font** - System sans-serif stack (`-apple-system, BlinkMacSystemFont, 'Segoe UI'...`)
- **Monospace Font** - SF Mono, Monaco, Cascadia Code for code and data
- **Base Size** - 14px body text with 1.5 line height
- **Headings** - Semibold (600 weight) for H1-H3

## Spacing & Layout

- **8px base unit** - All spacing increments follow 8px grid
- **Spacing scale** - 0.5x (4px), 1x (8px), 2x (16px), 3x (24px), 4x (32px)...
- **Generous whitespace** - Data-first approach keeps UI chrome minimal

## Components

All components are built with:

- **Shadcn-ui** (New York style) as foundation
- **Tailwind CSS** for utility-first styling
- **Exact Hex colors** extracted from visual references
- **Accessible interactions** - Focus rings, keyboard navigation

## Quick Start

```tsx
import { Button, Card, Badge } from '@olympus/ui';

// Primary button with Hex gradient
<Button className="bg-gradient-to-r from-blue-500 to-blue-600">
  Run Query
</Button>

// Card with hover shadow
<Card className="hover:shadow-md">
  <h3 className="text-lg font-semibold text-gray-900">Data Analysis</h3>
  <Badge className="bg-blue-500 text-white">SQL</Badge>
</Card>
```
